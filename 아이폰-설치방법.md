# 아이폰에 "일상 기록" 앱 설치하기

이 폴더(`iphone-app`)가 아이폰용 앱 전체예요. 데이터는 전부 아이폰 안에만 저장되고, 인터넷에 올라가는 건 앱 화면(코드)뿐이에요.

## 1. 앱을 인터넷에 올리기 (한 번만, 약 10분)

아이폰이 오프라인에서도 앱을 쓰려면 HTTPS 주소가 한 번 필요해요. GitHub Pages가 무료이고 제일 간단해요.

1. [github.com](https://github.com) 가입(또는 로그인)
2. 오른쪽 위 **+ → New repository** → 이름을 `daily-log-app` 정도로 짓고 **Public**으로 생성
3. 만들어진 저장소 페이지에서 **uploading an existing file** 링크 클릭
4. 이 폴더의 파일 6개를 전부 드래그해서 업로드 → **Commit changes**
   - `index.html`, `sw.js`, `manifest.webmanifest`, `icon-192.png`, `icon-512.png`
   - (`맥-기록-내보내기.json`은 올리지 마세요 — 내 기록이 공개돼요!)
5. 저장소의 **Settings → Pages** → Branch를 `main`으로 선택하고 **Save**
6. 1~2분 뒤 `https://내아이디.github.io/daily-log-app/` 주소가 생겨요

> 공개 저장소가 부담되면: 코드만 공개되는 거라 기록 내용은 절대 안 올라가요. 그래도 싫으면 Cloudflare Pages(무료, 비공개 가능)도 방법이에요.

## 2. 아이폰에 설치하기 (1분)

1. 아이폰 **Safari**에서 위 주소 접속
2. 공유 버튼(□↑) → **홈 화면에 추가**
3. 홈 화면에 생긴 "일상 기록" 아이콘으로 실행

이제 비행기 모드에서도 동작해요. (제목 자동 가져오기만 인터넷이 필요)

## 3. 맥의 기존 기록 가져오기 (선택)

1. `맥-기록-내보내기.json` 파일을 아이폰으로 보내기 — AirDrop이 제일 쉬워요 (파일 앱에 저장)
2. 아이폰 앱에서 오른쪽 위 **⚙︎ → 가져오기 (JSON)** → 그 파일 선택
3. "합치기" 선택

## 알아두면 좋은 것

- **백업**: ⚙︎ → 내보내기를 가끔 눌러서 JSON을 파일 앱이나 맥에 저장해 두세요. 앱을 지우면 데이터도 같이 사라져요.
- **맥 ↔ 폰 동기화**: 자동 동기화는 없어요. 양쪽 다 내보내기/가져오기(합치기)로 맞출 수 있어요.
- **앱 수정**: `index.html`을 고쳐서 GitHub에 다시 업로드하면 돼요. 그리고 `sw.js` 맨 위 `dailylog-v1`을 `v2`로 올려야 폰에 새 버전이 깔려요.
